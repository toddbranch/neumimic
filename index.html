Introducing Neumimic

Neumimic is an automated physical therapy system based around the X-Box Kinect. Doctors and therapists can prescribe exercises and patients can be guided through any number of repetitions through the system. Check out the video!

Have a Kinect and power supply? Try it yourself!
  1. Download Neumimic!
  2. Download and install the free Kinect SDK!
  3. Power the Kinect and plug it into your computer!
  4. Run Neumimic!

Supported operating systems: Windows 7, Windows 8.
